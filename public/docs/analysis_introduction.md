## Analysis submitting

TIMEDB provides a friendly submission page to guides users across the whole task submission procedure step by step.

Users could view detailed analysis tutorial as well as running demo and viewing demo results with one click.

<img src="../data/image/tutorial/a-1-1ppt.png" style="width: 60%;">

Task submission procedure has two parts, file uploading or TIMEDB dataset choosing, and parameter setting. In particular, users can upload their own files for analysis, or they can choose one dataset available in TIMEDB. We provide detailed descriptions for inputs and parameters as well as demo files to help users get familiar with input files format. Please be noted that uploaded files should contain at least two samples.
